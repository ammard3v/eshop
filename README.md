# 🛍️ Full-Stack eCommerce Application

![App Screenshot](app_sm.png)

Excited to share my latest project!

Over the past **5 months**, I independently developed a **full-stack eCommerce application**, focusing primarily on the Flutter frontend while building a complete backend and admin system.


## 1) User App (Flutter)

- Clean Architecture + BLoC  
- GoRouter navigation  
- GetIt for dependency injection  
- Dio with interceptors (auto access & refresh token handling)  
- Deep linking support (email verification flow)  
- Payment integration via Stripe  

## 2) Backend (Node.js)

- RESTful APIs  
- Authentication & authorization  
- Product & order management  
- Payment handling  

## 3) Admin Panel (React)

- User management (enable/disable accounts)  
- Product CRUD operations  
- Order tracking & status updates  
- Send targeted push notifications  
- *(Admin dashboard UI was prepared in collaboration with a friend)*  


##  #Key Features

- Firebase Cloud Messaging (FCM) for real-time push notifications  
- Order status updates (e.g., delivered alerts)
